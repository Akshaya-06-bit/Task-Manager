# 📝 Task Manager App | Full-Stack MERN Project

A modern and responsive **full-stack Task Manager web application** built with **React, Node.js, Express, and MongoDB** that allows users to securely manage their daily tasks with **authentication** and complete **CRUD functionality**.

> 🚀 Built to practice **React**, **REST APIs**, **MongoDB**, **authentication**, and **full-stack development** in a real-world project structure.

## 📌 Project Overview

This project is a **full-stack Task Manager App** where users can:

- 🔐 Register and log in securely
- ➕ Create new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 📋 View and manage their own tasks in a clean UI

This project helped me strengthen my understanding of:

- React fundamentals
- Frontend-backend integration
- REST API development
- MongoDB database operations
- Authentication and protected routes
- Full-stack application structure
